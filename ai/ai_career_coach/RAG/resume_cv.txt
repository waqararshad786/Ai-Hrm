# RESUME & CV GUIDE FOR PAKISTAN

## ATS-Friendly Resume Structure

1. Contact Information:
- Name (bold, large font)
- Phone (+92 3XX XXXXXXX)
- Email (professional, e.g., first.last@gmail.com)
- LinkedIn profile (custom URL)
- GitHub profile (for developers)
- Portfolio website (if applicable)
- Location (City, Pakistan)

2. Professional Summary:
- 2-3 lines summary
- Years of experience
- Key skills and technologies
- Top achievements
- Career objective
- Industry focus (e.g., Fintech, E-commerce)

3. Work Experience:
- Company name
- Position title
- Dates (Month Year - Present)
- 3-5 bullet points with achievements
- Use action verbs: Developed, Led, Created, Improved, Optimized
- Include metrics (e.g., increased performance by 40%)
- Mention technologies used

4. Education:
- Degree name
- Institution
- Year of graduation
- CGPA (if above 3.0)
- Relevant coursework
- Thesis/Project title

5. Skills Section:
- Technical skills (categorized)
- Tools & Technologies
- Languages (English, Urdu, etc.)
- Soft skills (Communication, Leadership)

6. Projects:
- 2-3 significant projects
- Technologies used
- GitHub links
- Live demo links
- Project description and impact

7. Certifications & Achievements:
- Relevant certifications
- Awards and recognition
- Publications
- Speaking engagements

## Common Mistakes to Avoid

1. Using fancy templates (ATS can't read complex formatting)
2. Including photos (not recommended in Pakistan)
3. Long paragraphs (use bullet points)
4. Spelling mistakes and grammatical errors
5. Missing contact information
6. Outdated skills listed
7. Too long (keep to 1-2 pages)
8. Generic objective statements
9. Not tailoring for each application
10. Missing LinkedIn and GitHub links
